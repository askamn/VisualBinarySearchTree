# VisualBinarySearchTree
A visual binary search tree to simulate how a binary search tree works when nodes/keys are inserted in it.

# Usage
The project was made with Eclipse Neon 4.6.1 
A compiled JAR package is also provided which you can run directly on a system having Java SE installed.
